Chess Notepad

Chess Notation and Clock

Easily save over the board chess games while mastering algebraic notation without trying.

How the Clock Works

Clocks are set as "N minutes | N increment seconds". The increment is added after each move.

The screen is divided in half. Black taps his or her clock to start the other clock and the game.

How to Notate Your Moves

This app is meant to guide you in notation while also teaching you how to do it. Each player will get a serious of buttons for notation once they've made their move and stopped their clock.

Step 1: Note the piece you moved

The first step is to note the kind of piece you moved. Tap the letter of the piece:

  • "K" = King
  • "Q" = Queen
  • "B" = Bishop
  • "N" = Knight
  • "R" = Rook
  • " " = Pawn

Knight's are known as N since K is reserved for the king. If a letter is not given that indicates a pawn. That's what the blank button is for.

NB: Castling requires you only tap one button, there are two button options availabe one for kingside castling and one for queenside.

Steps 2 & 3: Note the square you moved to

The board is composed of files (columns) and ranks (rows). They are both named from white's perspective. The file with white's leftmost rook is "a", the next one is "b" and so on until "h". The rank where white's major pieces start is "1", white's pawns start on "2" and so on. This arrangment means that black's leftmost rook starts on the h file and the 8th rank. In this system, each square has a name such as f4 or b8.

In most cases, noting the piece and the square is all you need to do.

Special Circumstances

However, some circumstances require more information.

Disambiguating Pieces

Depending on how the board is arranged, the same type of piece may legally be able to move to the same square. For example, if both white rooks are on the first rank with no pieces between them, either rook may move anywhere between them. In this instance you would need to specify the square the rook came from as well as its destination square.

You only need to provide either the file or rank of the starting square since that is almost always enough to disambiguate. However sometimes you need to specify both rank and file and it never hurts to specify both. You can do this by pressing "Note Starting Position" after you've indicated which piece you've moved.

NB: When a pawn captures, you must always note its starting position, even if it couldn't be confused with another pawn. This is just the way things are.
Pawn Promotion

When a pawn reaches the other end of the board it will get promoted. In this case, you'll be presented buttons for you to indicate which peice the pawn was promoted to (usually a queen).
